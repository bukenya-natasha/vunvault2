<div class="container">

<div class="step">
<h2>Step 1 – Download OWASP ZAP</h2>

<p>
Download OWASP ZAP from the official website:
</p>

<p>
<a href="https://www.zaproxy.org/download/" target="_blank">
https://www.zaproxy.org/download/
</a>
</p>

<div class="tip">
Install the latest stable version. If possible, choose the installer that includes Java.
</div>

</div>

<div class="step">

<h2>Step 2 – Install Java</h2>

<p>
If ZAP asks for Java, install Java 17 or later.
</p>

<p>
<a href="https://adoptium.net/" target="_blank">
https://adoptium.net
</a>
</p>

</div>

<div class="step">

<h2>Step 3 – Launch OWASP ZAP</h2>

<p>

When ZAP opens:

</p>

<ul>

<li>Select <strong>No, I do not want to persist this session.</strong></li>

<li>Click <strong>Start</strong>.</li>

</ul>

</div>

<div class="step">

<h2>Step 4 – Use a Safe Practice Website</h2>

<p>

Never scan websites without permission.

Use intentionally vulnerable practice applications such as:

</p>

<ul>

<li>OWASP Juice Shop</li>

<li>DVWA</li>

<li>bWAPP</li>

</ul>

<div class="warning">

Only scan systems that you own or have explicit permission to test.

</div>

</div>

<div class="step">

<h2>Step 5 – Start an Automated Scan</h2>

<ol>

<li>Open Quick Start</li>

<li>Select Automated Scan</li>

<li>Enter the target URL</li>

<li>Click Attack</li>

</ol>

</div>

<div class="step">

<h2>Step 6 – Spidering</h2>

<p>

ZAP first crawls the application.

It discovers:

</p>

<ul>

<li>Pages</li>

<li>Forms</li>

<li>Buttons</li>

<li>Directories</li>

<li>Links</li>

</ul>

</div>

<div class="step">

<h2>Step 7 – Active Scan</h2>

<p>

After crawling, ZAP begins checking for common security issues including:

</p>

<ul>

<li>Cross-Site Scripting (XSS)</li>

<li>SQL Injection</li>

<li>Security Headers</li>

<li>Cookie Security</li>

<li>Authentication Issues</li>

<li>Information Disclosure</li>

</ul>

</div>

<div class="step">

<h2>Step 8 – Review Alerts</h2>

<p>

Click the Alerts tab to review findings.

</p>

<ul>

<li>🔴 High</li>

<li>🟠 Medium</li>

<li>🟡 Low</li>

<li>🟢 Informational</li>

</ul>

<p>

Each alert contains:

</p>

<ul>

<li>Description</li>

<li>Risk</li>

<li>Recommendation</li>

<li>Reference</li>

</ul>

</div>

<div class="step">

<h2>Step 9 – Generate a Report</h2>

<p>

After the scan:

</p>

<div class="code">

Report → Generate Report

</div>

<p>

Choose HTML or PDF and save your results.

</p>

</div>

<div class="step">
